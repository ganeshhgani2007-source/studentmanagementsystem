# Student Management System (C++)

## Description
This project is a simple Student Management System built using C++.
It allows users to manage student records using a menu-driven program.

## Features
- Add Student
- View Students
- Search Student
- Delete Student
- File storage using students.txt

## Concepts Used
- Structures
- Functions
- File Handling
- Menu-driven programming

## Project Files
main.cpp – contains the main menu  
student.h – structure and function declarations  
student.cpp – implementation of functions  
students.txt – stores student records

## How to Run

Compile the program:

g++ main.cpp student.cpp -o student

Run the program:

./student
